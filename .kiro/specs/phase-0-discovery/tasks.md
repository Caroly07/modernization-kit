# Phase 0: Discovery Interview — Tasks

## Tasks

- [ ] 1. Conduct Organization & Industry interview
  Ask about core business, industry, size, location, cloud accounts, IT budget.

- [ ] 2. Conduct Regulatory & Compliance interview
  Identify all applicable frameworks. Ask about compliance team, recent audits, data residency, classification.

- [ ] 3. Conduct Application Landscape interview
  Inventory all applications with tech stack details. Map inter-app communication and dependencies.

- [ ] 4. Conduct Initial Pain Points interview
  Document top frustrations, workarounds, deployment process, backup status, security incidents.

- [ ] 5. Conduct People & Knowledge interview
  Identify original developers, current team, documentation status, institutional knowledge, modern tech experience, AI/OSS policies.

- [ ] 6. Conduct Infrastructure Inventory
  Catalog compute, storage, networking, GPU hardware, software licenses, and underutilized resources.

- [ ] 7. Conduct Goals & Constraints interview
  Capture drivers, success criteria, timeline, budget, downtime tolerance, mandates, deadlines.

- [ ] 8. Produce DISCOVERY_SUMMARY.md
  Compile all findings into a structured document with clear sections for answered questions, open questions, and implications.

- [ ] 9. Human review checkpoint
  Present Discovery Summary to domain expert and stakeholders for review and correction. Update based on feedback.
